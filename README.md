# iOSDS

Mini homepage for a music App

## Installation 

1. Install CocoaPods 
```
  sudo gem install cocoapods
  pod setup --verbose 
```
2. Install Pods in your Xcode Project
```
  pod init 
  ```
  add the following to your pod file: 
```
  pod 'Alamofire'
  pod 'Kingfisher', '~> 5.12'
  pod 'ObjectMapper'
 ```  
 Save the file and do ```pod install``` in your terminal
